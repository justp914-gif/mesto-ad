## Команды

- `npm run dev` — локальный запуск проекта
- `npm run build` — сборка проекта
- `npm run deploy` — деплой собранного проекта

## Деплой

Автоматический: при пуше в `main` через GitHub Actions в репозиторий [mesto-production](https://github.com/justp914-gif/mesto-ad).

Ручной: `npm run deploy`.
